Microsoft 365 for Beginners

Microsoft 365 is a huge cloud-based productivity tool that can be overwhelming for beginners. In this series of articles, we break down Microsoft 365 into easier, digestible modules for you. Go from being a beginner to an advanced Microsoft 365 user within a matter of days!

Microsoft 365 How Tos

Here you will find Microsoft 365 related how to articles.These articles contain concise explanations, supported by lot of click-and-play gifs that make it easier for the reader to understand what is being explained. With these articles we aim to cover every how to related to Microsoft 365 - be it apps, user adoption, or admin tasks.

Microsoft 365 Graph API

This corner is for newbie Microsoft 365 Developers. Especially those who are interested in creating simple apps that extend Microsoft 365 functionality by making use of Microsoft 365 APIs. Learn how to register your app, generate the API key, apply for app permissions and more here.

Microsoft 365 Updates

This corner is for the latest news and updates concerning Microsoft 365. The more updated you are, the better, as these updates help you get the best of out your Microsoft 365 tenant and help your organization become more productive.

Microsoft 365 Administration

This corner is for newbie Microsoft 365 administrators. It covers different topics like the various admin centers available within Microsoft 365, admin roles and responsibilities, Microsoft 365 user management, Microsoft 365 tenant security management and much more.
