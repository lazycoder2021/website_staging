Microsoft 365 Admin Centers

Microsoft 365 Admin Centers are web-based portals that provide administrators with tools and interfaces to manage and configure various aspects of their Microsoft 365 subscription. Microsoft 365, formerly known as Office 365, is a cloud-based suite of productivity tools that includes applications like Microsoft Word, Excel, PowerPoint, as well as services such as Exchange Online, SharePoint Online, and Teams. The Admin Centers serve as centralized hubs for administering and controlling these services. Here are some of the key Microsoft 365 admin centers.

Microsoft 365 Admin Center

The primary hub for managing Microsoft 365 services and users. It provides access to various administrative functions, including user management, license assignment, group management, role management, billing related activities, organization settings and more.

Microsoft 365 Admin Center URL: https://admin.microsoft.com

Exchange Admin Center

This admin portal is for specifically for managing Exchange Online, which is the cloud-based email and calendaring service in Microsoft 365. Administrators can configure mailboxes, set up mail flow rules, and manage security and compliance settings here.

Exchange Admin Center URL: https://admin.exchange.microsoft.com/

Exchange Admin Center blade under Microsoft 365 Admin Center.

Note: Exchange Admin Center (EAC) can also be accessed by selecting the Exchange Admin Center blade under Microsoft 365 Admin Center.

SharePoint Admin Center

Used for managing SharePoint Online, which is a cloud-based collaboration platform that allows Microsoft 365 users to collaborate and work together. Administrators can configure site collections, set up sharing and access policies, and manage content.

SharePoint Admin Center URL: https://[your-domain-name]-admin.sharepoint.com/

SharePoint Admin Center blade under Microsoft 365 Admin Center.

Note: SharePoint Admin Center (EAC) can also be accessed by selecting the SharePoint Admin Center blade under Microsoft 365 Admin Center.

Teams Admin Center

Helps administrators manage Microsoft Teams, the collaboration platform that integrates chat, video conferencing, and file sharing. Administrators can control Teams settings, manage users and guest access, and configure Teams-related policies.

Teams Admin Center URL: https://admin.teams.microsoft.com

Teams Admin Center blade under Microsoft 365 Admin Center.

Note: Teams Admin Center can also be accessed by selecting the Teams Admin Center blade under Microsoft 365 Admin Center.

Security Admin Center

Helps administrators manage email security, document security, app security, configure security policies and rules, track security related incidents, create/trigger security alerts, investigate security incidents, view security reports and much more

Security Admin Center URL: https://security.microsoft.com/

Security Admin Center blade under Microsoft 365 Admin Center.

Note: Security Admin Center can also be accessed by selecting the Security Admin Center blade under Microsoft 365 Admin Center.

Compliance Admin Center

Compliance Admin Center is a centralized hub within the Microsoft 365 ecosystem designed to help organizations manage and maintain compliance with various regulations and standards. Here are some of the key functionalities of the Compliance Admin Center: (i) Data Loss Prevention (DLP), (ii) Information Governance, (iii) Communication Compliance, (iv) Insider Risk Management and more.

https://compliance.microsoft.com

Compliance Admin Center blade under Microsoft 365 Admin Center.

Note: Compliance Admin Center can also be accessed by selecting the Compliance Admin Center blade under Microsoft 365 Admin Center.

Endpoint Manager (Intune) Admin Center

This admin center is used for managing and securing devices (such as PCs, laptops, and mobile devices) that connect to Microsoft 365. Administrators can configure policies, deploy applications, and ensure device compliance here.

Intune Admin Center URL: https://intune.microsoft.com/

Intune Admin Center blade under Microsoft 365 Admin Center.

Note: Intune Admin Center can also be accessed by selecting the Endpoint Manager Admin Center blade under Microsoft 365 Admin Center.

Identity Admin Center (Microsoft Entra ID)

Microsoft Entra ID is Microsoft's cloud-based identity and access management (IAM) service, widely used for managing Microsoft 365 user identities and access to resources in the Microsoft 365 ecosystem. The Microsoft Entra ID Admin Center offers various functionalities aimed at simplifying identity and access management tasks. Here are some key functionalities: (i) Identity Protection, (ii) Access Control, (iii) User Management and (iv) Group Management.

Identity Admin Center URL: https://entra.microsoft.com/

Identity Admin Center blade under Microsoft 365 Admin Center.

Note: Identity Admin Center can also be accessed by selecting the Endpoint Manager Admin Center blade under Microsoft 365 Admin Center.
